Regras da NLW Valoriza

Cadastro de Usuário
    Não é permitido cadastrar + de 1 usuário com o mesmo e-mail.
    Não é permitido cadastrar usuário sem e-mail.
Cadastro de Tag
    Não é permitido cadastrar tag sem nome.
    Não é permitido cadastrar + de 1 tag com o mesmo nome
    Não é permitido o cadastro por usuários que não sejam admin.
Cadastro de Elogios
    Não é permitido um usuário cadastrar um elogio para si.
    Não é permitido cadastrar elogios para usuário inválidos.
    O usuário precisa estar autenticado na aplicação.
